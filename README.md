🔤 Anagram Solver – Анаграммы (составь слово) на 7 языках
Классическая игра в анаграммы с таймером, подсказками, уровнями сложности и таблицей рекордов.
Составьте слово из перемешанных букв, тренируйте мозг и словарный запас!
Реализована на 7 языках программирования с единым интерфейсом и расширенными возможностями.

🚀 Возможности
Генерация анаграмм – случайное слово из словаря перемешивается, игрок должен восстановить оригинал.

Три уровня сложности:

Лёгкий – слова длиной 3–5 букв, время на раздумье 60 секунд.

Средний – слова длиной 5–7 букв, время 45 секунд.

Сложный – слова длиной 7–10 букв, время 30 секунд.

Подсказка – открыть первую букву (снимает 5 очков).

Система очков – за каждое угаданное слово начисляются очки (в зависимости от длины слова).

Таймер – ограничение на каждое слово.

Статистика – количество угаданных слов, пропусков, общий счёт.

Сохранение рекордов – лучший результат сохраняется в файл.

Цветной вывод – выделение букв, сообщений и таймера.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [уровень] [опции]
Уровни
easy – лёгкий (по умолчанию).

medium – средний.

hard – сложный.

Опции
Опция	Описание
-s, --stats	Показать статистику и выйти
-r, --reset	Сбросить рекорды
-h, --help	Справка
Управление
Введите слово целиком и нажмите Enter.

? – подсказка (открывает первую букву).

pass – пропустить слово.

quit – выход.

Примеры
bash
# Лёгкий уровень
python anagram.py

# Средний уровень
python anagram.py medium

# Показать статистику
python anagram.py -s

🧠 Логика игры
Выбор слова – случайное слово из встроенного словаря в зависимости от уровня.

Перемешивание – буквы перемешиваются, пока не получится слово, отличное от оригинала.

Отображение – игрок видит перемешанные буквы и таймер.

Ввод – игрок вводит свой вариант.

Проверка – если слово совпадает с оригиналом, начисляются очки.

Подсказка – показывает первую букву.

Пропуск – можно пропустить слово (без штрафа, но без очков).

Счёт – очки начисляются за длину слова (количество букв).

Игра – продолжается до тех пор, пока игрок не решит выйти или не будет достигнут лимит слов (по умолчанию 10).

✨ Дополнительные фичи
Автосохранение – статистика сохраняется в файл ~/.anagram_stats.json.

Цветной вывод – таймер, буквы, сообщения выделены.

Гибкий словарь – можно легко расширять словарь в коде.
